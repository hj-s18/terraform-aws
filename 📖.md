# Region

리전(Region) <br>
- 물리적으로 분리된 데이터 센터들의 모음 <br>
- 각 Region은 독립적임 (⇒ 한 Region 장애 발생 시 다른 Region 영향 없음) <br>
- 글로벌 서비스를 운영할 때 여러 Region 활용 가능 <br>

# 서울 리전에 VPC 생성

서비스의 속도 측면에서는 가까운 리전이 좋지만, <br>
속도가 중요하지 않다면 가격 측면을 고려하여 다른 리전을 선택하는 것도 방법임 <br>
