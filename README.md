# VPC 생성

VPC 이름 : tf_vpc <br>
IPv4 CIDR 수동 입력 : 10.0.0.0/16 <br>
IPv6 CIDR 블록 : 없음 <br>
테넌시 : 기본값(default) <br>
<br>
퍼블릭 엑세스 차단 : 비활성 (이건 VPC 설정에서 해주는 것 아님) <br>
DNS 확인 : 활성화됨 <br>
DNS 호스트 이름 : 비활성화됨 <br>
<br>
⇒ 자동으로 생성 : 네트워크 ACL, default 보안 그룹, local 라우팅 테이블, DHCP 옵션 세트
